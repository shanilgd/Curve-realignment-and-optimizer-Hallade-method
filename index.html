User Manual

Hallade Curve Realignment

1. Enter the number of stations before, on, and after the curve.

2. Click Generate Grid to create the input table.

3. Input the Existing Versines and optionally Proposed Versines.

4. Specify any inward/outward slew limits for specific stations.

5. Click Optimize Slew to automatically calculate the best realignment.


Note: You can also import data directly using the Import Excel button using the provided template.
